# AI Career Success Predictor

## Overview
AI Career Success Predictor is a Machine Learning web application that predicts whether a student is likely to get placed based on academic performance, skills, internships, projects, and other factors.

## Features
- Placement Prediction
- Placement Probability
- Placement Readiness Score
- Career Recommendations
- Skill Gap Analysis
- Interactive Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Classifier
- Streamlit
- Joblib

## Input Features
- Student ID
- Age
- Gender
- Degree
- Branch
- CGPA
- Internships
- Projects
- Coding Skills
- Communication Skills
- Aptitude Test Score
- Soft Skills Rating
- Certifications
- Backlogs

## Output
- Placement Prediction
- Placement Probability
- Readiness Score
- Career Suggestions
